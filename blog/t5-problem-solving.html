Problem Solving

"Instead of thinking outside the box, get rid of the box." -Deepak Chopra

Tuesday, 12 April 2016

That time I got blocked on a simple problem.

I couldn't figure out why my loop wasn't working. I was trying to get it to check if my element had a class of "active", and if it did to set it to "". I tried running the loop manually and using console.log. In the end I had to ask a member of my cohort who was able to tell me I was comparing a class instead of assigning a new class. It was so frustrating that my coded didn't work because of a simple mistake. I felt silly, but I make sure I check my assignment and comparison operators thoroughly now.

That time I solved a problem in an elegant way.

I'm really not at the stage where I feel like I'm a confident problem solver. I feel like I always end up with the longest most difficult way of solving something.

Reflect on how confident you feel using the problem solving techniques and process:

  • Pseudocode: 2/5
  • Trying something: 5/5
  • Rubber ducky method: 1/5
  • Reading error messages: 2/5
  • Console.logging: 2/5
  • Googling: 4/5
  • Asking your peers for help: 2/5
  • Asking coaches for help: 1/5
  • Improving your process with reflection: 2/5

I think can write pseudocode, but the problem I have is translating my pseudocode to real code. I think my problem is knowing where to start. I spend a lot of my coding time talking to myself :( I would ask for help, but most of the time, I'm not even sure what I need help with when it's everything. Yes, I know I should be starting small. I just find that I'll start on something small, it will work, but I have trouble figuring out how to tie it to the rest of my programme.

Using the JS Array docs as a starting point, describe what these functions do in your own words:

.map()
This method will go through each item in the array and call a function on it. It will then return a new array with the results.
.filter()
This method will filter out the items in the array as specified by a function and create a new array with the filtered items that return true.
.reduce()
This method is used to go through an array and condense it into one value.
