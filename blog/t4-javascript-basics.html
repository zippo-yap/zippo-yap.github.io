How does JavaScript compare to HTML and CSS?

If we stick with our person analogy, if HTML is the skeleton and CSS is equivalent to your visual features and clothing, then Javascript is similar to your nervous system and is responsible for interactions between your body and everything else. Javascript tells the browser how to change the webpage in response to things that happen (like clicking on something, or changing the value in a form input). In summary, javascript is a programming language that allows web developers to design interactive sites.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

A day in the life of a cat:

  1. Is cat hungry?
    • If no, go to sleep
    • If yes, go to question 2
  2. Is owner asleep?
    • If yes, go sit on their head
    • Go back to question 2
    • If no, meow very loudly. Go to question 3
  3. Is owner feeding cat?
    • If yes, eat food then go to sleep
    • If no, meow very loudly. Go to question 3

Repeat every time cat is awake.

Explain the difference between accessing data from arrays and object literals.

Arrays have order, but objects don't. Data from an array is accessed using a numerical index e.g. array[1], while data from object literals are accessed using bracket or dot notation e.g. object.property or object["property"]. You can also store a property in a variable and therefore use object[variable].

Explain what functions are and why they are useful.

A JavaScript function is a block of code designed to perform a particular task. This is where the fun comes in because you can re-use code to perform a task just by calling it. It removes the need to repeat all that lengthy code!
