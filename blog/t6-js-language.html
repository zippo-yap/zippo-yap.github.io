Javascript Language

"Learning another language is like becoming another person." — Haruki Murakami

Sunday, 24 April 2016

parantheses()
1. Execute a function - var result = doSomething()
2. Define a condition - if(condition){}
3. Define a parameter(s) - function(x,y){}
4. Hold an argument(s) - function(3,7)
brackets []
1. Hold an array - var array = [1,2,3]
2. Access/modify an index of an array - array[index]
3. Access/modify a key or property in an array - array[key]
4. Bracket notation to access a property of an object - object["property"]
braces {}
1.Holds objects
2.Encloses javascript statements
2.Group JavaScript statements together in code blocks
single quotes '' & double quotes " "
1.Define a string -you can use single or double quotes
2.Define a key in an object
